## Overview

**SOConstantsGenerator** is a constants generator for ScriptableObjects.

## Installation

To install, paste the following URL into Unity's **Package Manager**:

1. Open **Package Manager**.
2. Click the **+** button.
3. Select **"Add package from git URL..."**.
4. Enter:

```bash
https://github.com/hoangtongvu/SOConstantsGenerator.git?path=/Assets/SOConstantsGenerator
```
